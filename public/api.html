Robust API for managing tasks with user authentication and CRUD operations

API Endpoints

All endpoints require authentication except for signup and login.

  • POST https://innovative-flow-production.up.railway.app/api/signup User registration
  • POST https://innovative-flow-production.up.railway.app/api/login User authentication
  • GET https://innovative-flow-production.up.railway.app/api/tasks Get all tasks for authenticated user
  • POST https://innovative-flow-production.up.railway.app/api/tasks Create new task
  • PUT https://innovative-flow-production.up.railway.app/api/tasks/:id Update existing task
  • DELETE https://innovative-flow-production.up.railway.app/api/tasks/:id Delete task

Data Models

User Model

Property Type Constraints
username String required, unique, minLength: 3
password String required, minLength: 6
createdAt Date auto-generated
updatedAt Date auto-generated

Task Model

Property Type Constraints
userId ObjectId required (ref: User)
title String required
description String optional
status String enum: ["pending", "in-progress", "done"], default: "pending"
createdAt Date auto-generated
updatedAt Date auto-generated
